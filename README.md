# retail-sales-performance-seasonal-impact.
Project Overview

This project presents an end-to-end retail data analytics workflow analysing transactional sales data from a UK-based online retailer. The analysis integrates external datasets such as weather conditions, public holidays, and inflation (CPI) to assess their influence on sales performance.                                                       

Objectives
1.Analyse revenue and order trends over time
2.Identify seasonal sales patterns
3.Evaluate the impact of weather, holidays, and inflation
4.Communicate insights through interactive dashboards

Data Sources

1. Online Retail Transactional Data
UK-based online retail store transactions
Time range: December 2009 to December 2011
https://archive.ics.uci.edu/dataset/502/online+retail+ii

2. UK Public Holidays Dataset
Official bank and public holiday dates from the UK government
https://www.gov.uk/bank-holidays

3. Monthly Average Weather Data
Climate data including temperature, rainfall, sunshine, and frost.
https://climate-themetoffice.hub.arcgis.com/

4. Consumer Price Index (CPI) Data
CPI series for the United Kingdom
https://fred.stlouisfed.org/tags/series?t=cpi;united+kingdom
