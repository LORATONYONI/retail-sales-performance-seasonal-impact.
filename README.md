# retail-sales-performance-seasonal-impact.
Project Overview

This project presents an end-to-end retail data analytics workflow analysing transactional sales data from a UK-based online retailer. The analysis integrates external datasets such as weather conditions, public holidays, and inflation (CPI) to assess their influence on sales performance.                                                            Objectives

Analyse revenue and order trends over time

Identify seasonal sales patterns

Evaluate the impact of weather, holidays, and inflation

Communicate insights through interactive dashboards
Data Sources

UK Online Retail Dataset: https://archive.ics.uci.edu/ml/datasets/online+retail

UK Weather Data: https://www.metoffice.gov.uk/research/climate/maps-and-data

CPI Data: https://www.ons.gov.uk/economy/inflationandpriceindices

UK Public Holidays: https://www.gov.uk/bank-holidays                                                                                                                                Tools & Technologies
